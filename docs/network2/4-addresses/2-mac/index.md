# MAC-Adresse
---

## MAC-Adresse

Die physikalische Adresse, auch Hardware-Adresse eines Computers – genauer gesagt: seiner Netzwerkschnittstelle – nennt man MAC-Adresse (das hat nichts mit Apple zu tun, sondern steht für *media access control*). Sie besteht aus 48 Bit, die man üblicherweise als Hexadezimalzahl mit 6×2 Stellen angibt, beispielsweise: `f8:63:3f:67:72:d6`. Jede MAC-Adresse muss weltweit eindeutig sein, sie wird schon bei der Herstellung der Netzwerkkarte fix eingebaut.

MAC-Adressen sind essentiell für die digitale Kommunikation, weil Datenpakete ausschliesslich von einer Netzwerkkarte an eine andere weitergegeben werden können, also von einer MAC-Adresse an eine andere.

Die MAC-Adressen werden je nach Betriebssystem unterschiedlich dargestellt. Windows nutzt einen Bindestrich `-` als Trennzeichen, während die meisten anderen Betriebssysteme das Doppelpunkt `:` verwenden.

## Herstellersuche

Netzwerkgeräte-Hersteller erhalten gewisse Blöcke von MAC-Adressen zugewiesen. D.h. anhand einer MAC-Adresse kann der Hersteller ermittelt werden. Unter der Adresse [https://macvendors.com/](https://macvendors.com/) ist die Datenbank öffentlich zugänglich. Die MAC-Adresse `f8:63:3f:67:72:d6` gehört beispielsweise einer **Intel**-Netzwerkkarte.

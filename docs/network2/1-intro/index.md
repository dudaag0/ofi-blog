## Webseite

Woraus besteht eine Webseite? Kann man das auch selber machen? Was braucht es dazu?

::: exercise
### :exercise: Webseite

1. Öffne Thonny und erstelle eine neue Datei
2. Tippe den folgenden Inhalt ins leere Fenster

```html
<html>
  <head></head>
  <body>
    Hallo Welt
  </body>
</html>
```

3. Speichere die Datei als `hallo.html` in deinem Informatikordner ab.
4. Doppelklicke die Datei im _Windows Explorer_ oder im _Finder_.

Du hast soeben eine Webseiten-Datei erstellt – Sie lässt sich im Browser anschauen. Kannst du sie erweitern? Vielleicht so, dass dein Name und deine Hobbies angezeigt werden?
:::

Wir wollen nun etwas tiefer in den Aufbau und die Struktur einer Webseite eintauchen.

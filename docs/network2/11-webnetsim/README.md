# WebNetSim[^1]
---

WebNetSim ist eine interaktive Online-Lernumgebung, die spielerisch die grundlegenden Komponenten eines Netzwerks und deren Zusammenspiel vermittelt. Lernende übernehmen dabei Rollen wie Internetanbieter oder Haushalte, bauen eigene Netzwerke auf und vernetzen sich miteinander. So wird anschaulich gezeigt, wie z.B. eine Webseite vom Webserver über einen Provider bis zum heimischen Browser gelangt.

Link zur Online-Lernumgebung:
- [WebNetSim](https://webnetsim.de/)

Link zum Leitfaden für Lehrpersonen:
- [Leitfaden WebNetSim](https://webnetsim.de/LeitfadenWebNetSim.pdf)

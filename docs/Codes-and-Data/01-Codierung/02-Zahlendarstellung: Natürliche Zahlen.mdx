Jedes dieser sog. **Stellenwertsysteme** ist gleich aufgebaut. Schaut man sich das Zehnersystem an, bemerkt man, dass jede Stelle mit einer 10er Potenz multipliziert wird.

$$\

### BinГӨr вҶ’ Dezimal

Die Umrechnung vom BinГӨrsystem ins Dezimalsystem erfolgt durch Ausmultiplizieren mit den entsprechenden Zweierpotenzen.

Beispiel: `1011`
<div className="slim-table no-header">

|              |       |       |       |       |
| -----------: | ----: | ----: | ----: | ----: |
| Zweierpotenz | $2^3$ | $2^2$ | $2^1$ | $2^0$ |

$$
\begin{aligned}

\end{aligned}$$
